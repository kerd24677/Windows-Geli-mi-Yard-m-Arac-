===============================
 SOFTWAR - Network Tanılama Araçları
===============================

📁 Dosya: softwar.bat
📌 Amaç: Temel ağ tanılama ve bağlantı testlerini hızlıca gerçekleştirmek.
🖥️ Gereksinimler: Windows 10/11 ve üzeri (CMD ortamı)

-------------------------------
🔧 Kullanım
-------------------------------

softwar.bat çalıştırıldığında aşağıdaki işlemlerden istediğiniz seçeneği kullanabilirsiniz:

[1] IP Adresi Görüntüleme
[2] DNS Sunucularını Listeleme
[3] Ping Testi
[4] Traceroute (İzleme)
[5] Port Kontrolü (Netstat)
[6] Tüm Ağ Bağlantılarını Listele
[7] Çıkış

Ve birçok özellik sizi bekliyor.

Komut satırında seçenek numarasını girmeniz yeterlidir.

-------------------------------
📌 Özellik Açıklamaları
-------------------------------

1️⃣ IP Adresi Görüntüleme
→ Mevcut ağ adaptörlerinin IP yapılandırmalarını gösterir.

2️⃣ DNS Sunucularını Listeleme
→ Bilgisayarınızın bağlı olduğu DNS sunucularını listeler.

3️⃣ Ping Testi
→ Belirttiğiniz IP adresine veya domaine ulaşım olup olmadığını test eder.

4️⃣ Traceroute
→ Hedefe giden veri paketlerinin hangi router'lardan geçtiğini gösterir.

5️⃣ Port Kontrolü (Netstat)
→ Bilgisayarınızda açık olan bağlantı noktalarını (portları) listeler.

6️⃣ Ağ Bağlantılarını Listele
→ Etkin ağ bağlantılarını ve durumlarını listeler.
